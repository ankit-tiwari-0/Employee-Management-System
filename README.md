# 🚀 Employee Management System (EMS)

A modern and responsive **Employee Management System** built using **React + Vite + Tailwind CSS**.
This app helps manage employee tasks efficiently with a clean UI and smooth user experience.

---

## ✨ Features

* 👤 User Login System
* 📊 Dashboard with task statistics
* 📝 Task Management (New, Completed, Accepted, Failed)
* 🎨 Modern UI with gradients & glass effects
* 📱 Fully Responsive (Mobile, Tablet, Desktop)
* ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Deployment:** Vercel

## 🌐 Deployment

This project is deployed using **Vercel**.

### Steps:

1. Push code to GitHub
2. Import project in Vercel
3. Set:

   * Build Command: `npm run build`
   * Output Directory: `dist`
4. Deploy 🚀
